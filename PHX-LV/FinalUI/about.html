HoneyWell Aerospace: Improving Flight Plan Routes Based on Big Data and AL/ML

Capstone Project at Arizona State University 2023-2024

About Our Project

  • Flight-planning is a multi-faceted task that spans across multiple domains, including those of logistics, finance, physics, mathematics, and academia.
  • Our project aims to leverage big data and AI/ML technologies to enhance flight planning, particularly in improving ETA metrics.
  • The project involves data cleaning, exploratory data analysis, feature engineering, and model building to produce accurate results.

Methodology

Data Collection
  • We started scraping data from a new web scraper for ETA information and a regression model for ETA prediction.
  • In our previous work, we obtained latitude, longitude, and other pertinent data for airports worldwide from The Global Airport Database, a freely available resource.
  • We converted this data from a TXT format to CSV and filtered it to include only airports within the USA.
  • These airport coordinates serve as crucial starting and ending points for flights, as FlightAware data lacks this information.
Our Model
  • Feed Forward Neural Network (3 Layers) with ReLU Activation Function and 3 input features (ATD, Day of the Week, Average Delay Takeoff), 1 output feature (ETA).
  • During training, the model learns from the training dataset to optimize its parameters and minimize prediction errors.
  • The output layer of our neural network is responsible for producing the predicted flight arrival times.
  • The output layer of our neural network is responsible for producing the predicted flight arrival times. It receives the processed data from the hidden layers and generates the final output, representing the predicted actual time of arrival for each flight.
Results
  • Average difference between predicted and actual arrival times is approximately 2.58 minutes, indicating model accuracy.
  • Mean Squared Error (MSE) or Average squared difference between predicted and actual values: Train - 8.58, Test - 9.81.
  • Mean Absolute Error (MAE) or Average absolute difference between predicted and actual values: Train - 2.06, Test - 2.58.
  • Ridge regression model achieves MSE: Train - 9.26, Test - 8.26, providing an alternative prediction approach.
